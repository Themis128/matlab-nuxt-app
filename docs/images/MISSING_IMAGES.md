## Quick Capture Script

You can use browser DevTools to capture screenshots:

1. **Open DevTools:** Press `F12`
2. **Open Command Palette:** Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
3. **Type:** "Capture full size screenshot"
4. **Select:** "Capture full size screenshot"
5. **Save:** The screenshot will download automatically

## Notes

- All images should be saved in `docs/images/` directory
- Use PNG format for screenshots (better quality)
- Keep file sizes reasonable (< 1MB per image)
- The README.md will automatically display these images once they're added
